# 多方软件证书生成管理

## 当前不同版本证书生成规则的概述
1.  **冲击动力学求解器（三翼）**: 硬件网卡地址  + 单位名称  + 授权时长 + 联系人 + 模块标识
2.  **冲击动力学前处理+冲击动力学后处理（迈曦）**: 自研license生成工具，通过Mac地址+主机名称+许可时间的形式，生成.lic授权文件
3.  **多体前后处理器（英特）**: 采购第三方license生成工具，通过Mac物理地址+许可时间
4.  **结构（安世）**: 采购第三方license管理工具，支持浮动license许可和本地license文件
5.  **浮动license管理**: 部署license服务器时配置许可文件中的主机名和使用的端口，MAC地址+许可时间
6.  **本地license文件**: MAC地址+许可时间
